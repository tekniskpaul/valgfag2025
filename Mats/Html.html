Html

Gå Tilbake

Html er et programmerings språk som er veldig bra til å lage nettsider med. Html har også c++ some du kan få med å skirve style med <> rundt.

Det går ikke ann å lage en nettside uten html det er svært viktig for at en nettside skal virke å ha html kode i den

Html står for Hyper Text Markup Language og det var laget av Sir Tim Berners-Lee

Du kan bruke CSS i Html kode som står for Cascading Style Sheets som gjør at du kan ha litt bedre kode med stilen av koden som gjør at du kan endre stilen til skrift mye lettere når du lærer det
